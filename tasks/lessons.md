# Lessons Learned

Institutional memory for AI agents. This file is updated automatically
after every correction from the user. Review at the start of each session.

## Format

Each entry follows this structure:

```
### YYYY-MM-DD — Short title

**Context**: What I was doing
**Mistake**: What I did wrong
**Lesson**: What I should do differently
**Rule**: Concrete rule for the future
```
